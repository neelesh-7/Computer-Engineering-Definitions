  1. Home

  3. RAM
Description

  RAM stands for Random Access Memory which temporarily stores data that the CPU would need to access quickly.
  RAM is a type of memory which is volatile, so it loses all its data when the computer is turned off. Very different to ROM (Read-Only Memory).
  RAM significantly affects the computer's performance and how quickly and effectively it can process data.
  You can find memory modules which have designated slots for the RAM.
  A computer system's RAM is very important for efficiency and capabilities of the computer.
