Denial of Service

  1. Home
  2. Definitions
  3. Denial of Service
Definition

  Denial of Service (DoS) is a cyber attack that makes a computer, network or service unusable for the users.
  The attacker usually targets the system by sending in a high volume of traffic which makes the system use its bandwidth, processing power, and/or memory. This makes the system slow down, unresponsive, and sometimes even crash. However, it is less harmful than viruses and worms because a denial of service attack can only temporarily disrupt a network service.
  There are different kinds of DoS attacks like:

  1. Flooding, which sends a massive amount of traffic to the target system, overwhelming its capacity to handle the requests.
  2. Application layer, which targets vulnerabilities in specific applications or services running on the target system.
  3. Distributed denial of service, which attacks botnets to flood it with traffic.

  DoS can stop businesses, create financial losses, damage reputation, and much more.
  You can protect yourself by having:

  1. Firewalls and intrusion prevention system, which helps filter and block malicious traffic.
  2. Traffic monitoring and analysis, which monitors tools to detect abnormal traffic patterns or sudden increases in traffic.
  3. Load balancing, which distributes network or application traffic across multiple servers.
  4. DoS mitigation, which uses specialized services or appliances.
  5. Regular patching and updates, which keeps systems and software up to date with the latest security patches.
