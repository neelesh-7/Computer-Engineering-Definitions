Cache

  1. Home
  2. Definitions
  3. Cache
Description

  Cache is a tiny, temporary storage device that shortens the time a computer system takes to retrieve data from the memory or another storage device.
  An example would be Internet Browsers that use cache to save multiple copies of recently accessed websites, images, etc. When the user goes back to those websites or images, the brower instantly receives the data from the cache.
  Cache essentially speeds up the time it takes to access data and instructions that are commonly used.
  A user can also delete cache from any device in the settings or in the app itself. (See Figure 3)
