Description

  CPU stands for Central Processing Unit, which is the main part of a computer or any other electronic device. A CPU follows through instructions set by different components and sets different processes for the computer. It is also referred to as the computer's "brain".
  To be more detailed, the CPU reads the orders that it is given from components like the memory, decodes them and then carries them out.
  A faster CPU does this whole process much faster, and it is measured in instructions per second. A faster CPU causes the software to move more swiftly and faster.
  There are many different types of CPUs which include those that are in laptops, servers, cell phones, and other devices like appliances at home.
  The CPU can be found on top of the motherboard in the electronical device and in a CPU socket with pins down.
